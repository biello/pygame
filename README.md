# pygame

### usage
first pull this repo to your local filesystem, then
``` 
pip install pygame 
(if you are in China, you can try: pip install pygame -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com)
cd pixel_art
python main.py
```
